Luna

The Luna project provides data on the moon that is suitable for both astronomical and astrological purposes.

Luna for your website

The area on the right shows the current state of the moon using the Luna widget. You can easily embed the Luna widget on your own site for free.

API

Luna offers a JSON API that allows you to query the state of the moon at a certain point in time, and also to find new and full moons.
